# Machine Learning From Scratch

This repository contains implementations of some core machine learning algorithms written from scratch in Python, without using libraries like scikit-learn.

The goal is to understand how these models actually work under the hood rather than just using them as black boxes.

---

## Models Implemented

- Linear Regression → [linear-regression.py](./linear-regression.py)  
- Logistic Regression → [logistic-regression.py](./logistic-regression.py)  
- K-Nearest Neighbors (KNN) → [knn.py](./knn.py)  
- Support Vector Machine (SVM) → [svm.py](./svm.py)  
- Neural Network → [neural-network.py](./neural-network.py)  

---

## What's Covered

- Gradient Descent  
- Cost/Loss functions  
- Basic optimization  
- Distance metrics (for KNN)  
- Backpropagation (for neural networks)  

---

## Utilities

Common helper functions are in [`utils.py`](./utils.py), including things like distance calculations and activation functions.
